SVM 01 — What is Space Vector Modulation?

Space Vector Modulation (SVM) is the method used in motor-drive inverters to produce smooth, efficient motor currents using only simple ON/OFF switches.

On this page 1) The core idea 2) Visual preview What we are NOT doing yet Next

The core idea

A 3-phase inverter used in motor drives can only switch its transistors fully ON or fully OFF. Because of this hardware limitation, it cannot directly generate an arbitrary voltage.

Space Vector Modulation works within this limitation by switching between a small number of valid inverter states defined by the hardware.

To reason about these states clearly, we represent each one as a vector in a two-dimensional plane called the αβ plane.

If you have seen the Clarke transformation before, the αβ plane will look familiar. If not, don’t worry — for now, think of it simply as a 2D diagram used to visualize three-phase voltages. You do not need to derive or calculate anything in the αβ plane yet.
